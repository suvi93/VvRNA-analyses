# VvRNA-analyses
Scripts used for analyses in the paper Evolutionary dynamics of dosage compensation and sex-biased gene expression in Morabine grasshopper Vandiemenella viatica

1. alignment.sh - bash script used for alignment of all transcriptome samples
2. pca.R - R script used for PCA analyses
3. deseq2.R - R script used for differential gene expression analyses
5. heatmap.R - R script used to generate heatmap of top 25 up & down-regulated differentially expressed genes per tissue classified by chromosome
6. sbg_xavstackedbar.R - R script used to generate figure 3 in the paper
