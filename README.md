# VvRNA-analyses
Scripts used for analyses in the paper Evolutionary dynamics of dosage compensation and sex-biased gene expression in Morabine grasshopper Vandiemenella viatica

1. alignment.sh - bash script used for alignment of all transcriptome samples
2. deseq2.R - R script used for differential gene expression analyses
3. PCA.R - R script used for PCA analyses
4. heatmap.R - R script used to generate heatmap of top 25 differentially expressed genes per tissue classified by chromosome
5. sbg_xavstackedbar.R - R script used to generate figure 3 in the paper
